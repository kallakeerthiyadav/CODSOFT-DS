# Movie Rating Prediction
## Task
Predict movie ratings using machine learning regression techniques.
## Dataset
Movie dataset containing:
- Votes
- Runtime
- Rating
## Tools Used
- Python
- Pandas
- Scikit-learn
## Steps
1. Loaded movie dataset
2. Removed missing values
3. Selected numeric features
4. Applied Linear Regression
5. Evaluated model using Mean Absolute Error
## Result
The model predicts movie ratings with reasonable accuracy.
