# Iris Flower Classification
## Task
Classify Iris flowers into different species using machine learning.
## Dataset
Built-in Iris dataset from Scikit-learn containing:
- Sepal length
- Sepal width
- Petal length
- Petal width
## Tools Used
- Python
- Scikit-learn
## Steps
1. Loaded Iris dataset
2. Split data into training and testing sets
3. Trained Logistic Regression model
4. Evaluated model accuracy
## Result
The model classifies Iris flower species with high accuracy.
